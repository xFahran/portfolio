Profil

Un développeur qui pense architecture, sécurité et maintenabilité.

Je viens du développement web et mobile, avec une approche très orientée backend, qualité logicielle et sécurité. Je sais produire une interface, mais je ne m'arrête pas au joli bouton : je m'intéresse surtout à ce qu'il déclenche, comment les données circulent, comment les erreurs sont gérées et comment éviter qu'une application devienne une dette technique ambulante.

Ce que j'apporte

Une vision pragmatique du développement : comprendre le besoin, choisir une architecture lisible, sécuriser les entrées, structurer les données et livrer un code exploitable. Je travaille avec une logique défensive, en gardant en tête les risques OWASP, les tests et la maintenance.

Mon positionnement

Full stack avec un vrai tropisme backend et applicatif. Je peux intervenir sur une API Node.js, une application WPF, une interface EJS, une base PostgreSQL ou MariaDB, un environnement Docker, ou une application C++ avec Qt. Bref, pas juste déplacer des divs comme si l'humanité n'avait pas déjà assez souffert.
